# GlobeHoppers v8.2 Work Package 6

Implements private timeline ordering through `reorder_private_trips`, a complete ordered UUID list, and `travel_maps.timeline_order_revision`.

## Boundaries

- Reorder is available only for authenticated cloud maps with writes enabled.
- The full map order is saved atomically.
- Stale saves are rejected with a GlobeHoppers-styled conflict popup.
- Batch Add remains disabled.
